HTML（超文本标记语言）包含众多标签，下面为你介绍不同类型下常用的 HTML 标签：

结构标签:
这类标签用于构建网页的整体结构。

<html > ：HTML 文档的根标签，所有其他 HTML 元素都要包含在其中。

<head>：包含文档的元数据，像字符编码、页面标题、引用的外部文件等信息。

<title>：定义网页在浏览器标签页上显示的标题。

<body>：包含网页中可见的内容，如文本、图片、链接等。

<header>：通常表示页面或页面中某个区域的头部，常包含标题、导航等。

<nav>：用于定义导航链接。

<main>：表示文档的主要内容，一个页面应该只有一个 <main> 元素。

<article>：表示一个独立的、完整的内容块，比如一篇文章、博客帖子等。

<section>：用于对文档中的内容进行分块，通常包含一个标题。

<aside>：表示与页面主要内容相关的侧边信息。

<footer>：通常表示页面或页面中某个区域的底部，常包含版权信息、联系方式等。

文本标签:
用于处理和展示文本内容。

<p>：定义段落。

<h1> - <h6>：定义不同级别的标题，<h1> 级别最高，<h6> 级别最低。

<br>：换行标签，用于在文本中强制换行。

<hr>：水平分隔线，用于在页面中创建一条水平线来分隔内容。

<b>：将文本设置为粗体，但不具备语义。

<strong>：表示重要的文本，通常也显示为粗体，具有语义。

<i>：将文本设置为斜体，但不具备语义。

<em>：表示强调的文本，通常显示为斜体，具有语义。

<u>：为文本添加下划线。

<s>：给文本加上删除线。

<small>：使文本变小。

<mark>：对文本进行高亮显示。

<del>：表示文本已被删除。

<ins>：表示文本是新插入的。

<sub>：将文本设置为下标。

<sup>：将文本设置为上标。

列表标签:
用于创建有序或无序列表。

<u>：定义无序列表，列表项通常用圆点标记。

<ol>：定义有序列表，列表项通常用数字标记。

<li>：定义列表中的每一项，可用于 <ul> 和 <ol> 中。

<dl>：定义描述列表，由术语和描述组成。

<dt>：定义描述列表中的术语。

<dd>：定义描述列表中术语的描述。

链接和图像标签:
用于在网页中添加链接和图像。

<a>：定义超链接，用于链接到其他网页、文件或页面内的特定位置。

<img>：用于在网页中插入图像。

表单标签:
用于创建用户输入表单。

<form>：定义 HTML 表单，用于用户输入数据。

<input>：用于创建各种输入字段，如文本框、密码框、单选框、复选框等。

<label>：为 <input> 元素定义标签，提高表单的可用性。

<select>：定义下拉列表。

<option>：定义 <select> 元素中的选项。

<textarea>：定义多行文本输入框。

<button>：定义一个按钮，可用于提交表单或执行其他操作。

表格标签:
用于创建表格。

<table>：定义表格。

<thead>：定义表格的表头部分。

<tbody>：定义表格的主体部分。

<tfoot>：定义表格的页脚部分。

<tr>：定义表格中的一行。

<th>：定义表格中的表头单元格。

<td>：定义表格中的数据单元格。
